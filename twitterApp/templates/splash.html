Welcome to Sam and Nadine's Twitter project for CIS 192!

This web application allows users to tweet what's on their mind to their feed, where other users can also see what they've tweeted. They can tag their tweets with hashtags by simply adding a # before the tag. The web application will automatically create a tag if it did not previously exist. You can also click on each tag under the "Hashtags" table to view all the tweets with that given hashtag. Users can also like tweets by pressing the like button. Each user can only like a tweet once.
